Outra coisa importante que foi mencionada pelo professor foi que se colocarmos o atributo "style" dentro da tag <strong> podemos modificar a cor
que o "negrito" de strong vai ficar.

O professor também menciona também a tag <img> e diz que a mesma precisa de um atributo para funcionar, que seria o "src". O src serve para

Resumo da parte 3 (final):

Professor falou das listas ordenadas e não ordenadas. Usamos <ol> para listas ordenadas (a que tem ordem por números)
e <ul> para listas não ordenadas (a que tem ordem por bolinhas). Para ambas as listas, sempre será necessário usar <li> para incluir cada
item na lista, não dá para apenas colocar uma delas (<ol> ou <ul> ) e escrever entre elas.

Exemplo errado:

<ol>

qualquer item

</ol>

Exemplo correto:
<ol>
<li>Pegar a água;</li>

<li>Abrir garrafa;</li>

<li>Beber a água.</li>
</ol>

Exemplo de lista ordenada:

  1. Pegar a água;
  2. Abrir garrafa;
  3. Beber a água.

A mesma coisa serve para <ul>:

  • Pegar a água;
  • Abrir garrafa;
  • Beber a água.

É também possivel abrir uma lista ordernada(ou não ordenada) dentro de um <li> e então será necessário
repetir todo o processo anterior (o do exemplo correto) dentro dessa sub-lista.

Resumo parte 3.2:

O tutor do curso também mencionou que é possível inserir links nos textos usando a tag <a> .
Quando se trata de usar a tag <a> deveremos usá-la da seguinte maneira:

<a href="URL">A palavra ou frase que vai receber o link do href</a>
Dessa maneira em que está vendo, se você clicasse na frase que recebeu o link, você iria ser redirecionado na mesma guia, não abriria uma nova guia.

Exemplo:
Ao clicar, você não abrirá uma guia à direita.

Para que o link que seja aberto em uma guia nova à direita você deverá usar dentro de <a> o atributo
target e no atributo você deverá escolher a opção "blank", que abrirá uma guia à direita.
Exemplo:
<a href="URL" target="_blank">O texto abrirá uma guia à direita.</a>
Veja a seguir:
Ao clicar, você abrirá uma guia à direita.

Ao usar a tag <a> você também consegue colocar mais um outro atributo chamado title. Este atributo vai fazer que quando você aponte o mouse no hipertexto, apareça alguma mensagenzinha dentro de
um balãozinho. Pode inclusive checar os outros links e apontar neles que você verá que nada aparece, mas aponta neste daqui:
Não clica ainda, presta atenção no balãozinho da setinha do mouse.
