Como escrevemos tudo isso dentro do editor?

Colocamos na parte onde ficam os itens opcionais cumulativos da seguinte maneira:

<form action="#" method="get">

<h2>Você selecionou: Pizza de Calabresa</h2>

<p>Quais opcionais você deseja?</p>

<input type="checkbox" name="opcional[]" value="queijo"> + queijo<br>

<input type="checkbox" name="opcional[]" value="calabresa"> + calabresa<br>

<input type="checkbox" name="opcional[]" value="cebola"> + cebola<br>

<input type="checkbox" name="opcional[]" value="azeitona"> + azeitona<br>

Pode reparar que no atributo name coloquei a variável "opcional" junto com "[]".

O professor explicou, que quando usamos "[]", estamos
mandando essa informação para o servidor de maneira completa.
Caso estivesse sem colchetes, apareceria apenas a última
informação que foi marcada pelo usuário no campo opcional.
Ex.:
Usuário marcou cebola, azeitona e calabresa. Caso não houvesse colchetes
apenas chegaria na sua linguagem back-end o ítem calabresa.
Por isso é sempre bom estudar esses pequenos detalhes nas aulas.

Agora vamos para os opcionais onde ou é um, ou é outro.

<p>Borda recheada?</p>

<input type="radio" name="borda" value="sim">Sim<br>

<input type="radio" name="borda" value="nao">Não<br>

<p>Bebida?</p>

<input type="radio" name="bebida" value="suco">Suco<br>

<input type="radio" name="bebida" value="refrigerante">Coca-Cola<br>

<input type="radio" name="bebida" value="pepsi">Pepsi<br>

<input type="radio" name="bebida" value="sprite">Sprite<br>

<input type="radio" name="bebida" value="agua">Água<br>

<input type="radio" name="bebida" value="nenhum">Nenhum<br>

<button type="submit">Enviar pedido</button><br>

Se repararmos no campo "Borda recheada?", os seus inputs possuem no atributo name a mesma variável "borda".
O que faz com que dê para marcar apenas um ou o outro. o mesmo serve para as bebidas, todos possuem a mesma variável.

Caso você não digitasse nada em name enquanto usa "radio", seria possível marcar várias opções, deixando de ser
"um ou outro", o qual seria esse o propósito em usar "radio" para "um ou outro".
